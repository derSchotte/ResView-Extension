# ResView v1.1.8 – Release Notes

**Veröffentlicht:** Mai 2026  
**VS Code Mindestversion:** 1.85.0

---

## Neuerungen in dieser Version

### Permanenter „Live Server :5500"-Button

Für die VS Code **Live Server**-Extension (Port 5500) gibt es jetzt einen festen Schnellzugriff-Button in der Toolbar. Er ist immer sichtbar — unabhängig davon, ob der Auto-Scan einen Server gefunden hat oder nicht. Ein Klick lädt sofort `http://localhost:5500`.

Der Button ist visuell vom Rest der dynamisch erkannten Server-Chips abgesetzt (lila Akzentfarbe statt grün), damit er als fester Shortcut erkennbar ist.

### Port 5500 in der Auto-Detection

Port 5500 wurde zusätzlich in die Liste der automatisch geprüften Ports aufgenommen. Läuft Live Server, erscheint er wie alle anderen erkannten Server auch als Chip mit ✓-Badge.

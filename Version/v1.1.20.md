# v1.1.20 – 2026-05-26

## Box model in CSS Rules panel + mutual exclusion

### Added
- **Three-column CSS Rules panel.** The panel body now contains: rule list (left) | Firefox-style box model (centre) | declared properties (right). Selecting a CSS rule populates the box model with any declared margin, border-width, and padding values; shorthand properties (`margin: 8px 16px`) are expanded to individual sides.
- **Panels are mutually exclusive.** Opening the Inspector panel closes the CSS Rules panel, and vice versa. Only one panel can be visible at a time.

### Changed
- Inspector button no longer auto-opens the CSS Rules panel on activation.
- CSS Rules panel button now hides the Inspector panel when toggled on.

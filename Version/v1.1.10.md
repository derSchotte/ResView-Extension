# ResView v1.1.10 – CSS Source File Badge Fix

## Bug fixes

### Inspector: source file badges now render correctly

- **Root cause**: filenames like `<style>` were inserted via `innerHTML` without
  HTML-escaping, so the browser silently swallowed the tag and nothing appeared.
  Fixed by escaping `<`, `>`, and `&` before injection.
- **Readability**: badge font size increased from 10 px → 12 px, weight set to
  600, padding enlarged (2 px 8 px), and border/background slightly more
  opaque — the green filename chips are now clearly legible.

## Files changed

| File | Change |
|---|---|
| `media/main.js` | Added `escHtml()` helper; use it when rendering `.insp-source-file` spans |
| `media/style.css` | Increased badge size, weight, and contrast |

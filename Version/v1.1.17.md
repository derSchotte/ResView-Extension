# ResView v1.1.17 – Automatische Server-Erkennung im Hintergrund

## Neuerung

ResView prüft jetzt alle **3 Sekunden** im Hintergrund, ob neue Dev-Server
gestartet wurden – auch während die Extension bereits offen ist.

- Wird ein neuer Server erkannt (z. B. nach `npm run dev`), aktualisieren
  sich die **Server-Chips** in der Toolbar automatisch.
- Ist die URL-Leiste noch leer, erscheint eine **VS Code-Notification** mit
  den Buttons **„Open"** und **„Dismiss"** – ein Klick lädt den Server sofort.
- Kein manueller Rescan mehr nötig.
- Das Polling stoppt automatisch, wenn die Extension geschlossen wird.

## Technische Details

- Recursiver `setTimeout`-Loop (kein `setInterval`) verhindert Überlappungen,
  falls ein Scan länger als 3 s dauert.
- Nur *neue* Ports (nicht bereits bekannte) lösen die Notification aus.

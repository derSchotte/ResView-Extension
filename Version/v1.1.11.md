# ResView v1.1.11 – Inspector Source Files: only real filenames

## Bug fix

Inline `<style>` blocks are no longer listed as a source in the Inspector
panel. Only external stylesheet filenames (e.g. `main.css`, `styles.css`)
are shown. If a rule comes from a `<style>` tag without an `href`, it is
silently skipped.

## Files changed

| File | Change |
|---|---|
| `src/proxy.ts` | Skip stylesheets without `href` in `getSources()` |

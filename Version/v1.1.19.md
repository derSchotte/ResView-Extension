# ResView v1.1.19 – Lucide Icons

## Neuerung

Alle Toolbar-, Tab- und Modal-Icons wurden durch **Lucide SVG Icons** ersetzt.
Das Lucide UMD-Bundle (`lucide.min.js`) liegt lokal in `media/` und wird mit
einem CSP-Nonce geladen — keine externe Abhängigkeit zur Laufzeit.

| Bereich | Icon |
|---|---|
| URL-Leiste Chevron | `chevron-down` |
| Rescan-Button | `refresh-cw` |
| Im Browser öffnen | `external-link` |
| Live Server Chip | `zap` |
| Tab: Phone | `smartphone` |
| Tab: Tablet | `tablet` |
| Tab: Desktop | `monitor` |
| Rotate-Button | `rotate-cw` |
| Gerät hinzufügen | `plus` |
| Gerät löschen | `trash-2` |
| Geräteinfo | `info` |
| Grid | `grid-2x2` |
| Ruler | `ruler` |
| Inspect | `crosshair` |
| Modal schließen | `x` |

## Technische Details

- `lucide.createIcons()` wird einmalig am Anfang der IIFE aufgerufen; alle
  `<i data-lucide="...">` Elemente im statischen HTML werden zu SVGs.
- Klassen und IDs (z. B. `id="urlChevron"`, `class="url-chevron"`) werden von
  Lucide auf das erzeugte `<svg>` übertragen — bestehende CSS-Transitions und
  JS-Refs funktionieren unverändert.
- Spinner-Animation zeigt jetzt auf `.spinning svg` statt `.spinning .icon`.

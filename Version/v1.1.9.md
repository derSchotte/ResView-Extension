# ResView v1.1.9 – CSS Source File Display

## What's new

### Inspector: CSS source file badges

When hovering over an element in Inspector Mode, the panel now shows which CSS
file(s) each rule comes from — displayed as small green badges to the right of
the selector tag.

- Badges show the filename (e.g. `styles.css`, `main.css`, `<style>`)
- Multiple source files are listed side by side when a selector is defined in
  more than one stylesheet
- The detection uses `document.styleSheets` + `element.matches()` inside the
  injected inspector script, so it works for any stylesheet loaded on the page
- Inline `<style>` blocks are labelled `<style>` instead of a filename

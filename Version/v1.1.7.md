# ResView v1.1.7 – Release Notes

**Veröffentlicht:** Mai 2026  
**VS Code Mindestversion:** 1.85.0

---

## Neuerungen in dieser Version

### Grid – einstellbare Zellengröße

Neben dem **„Grid"-Button** gibt es jetzt ein Dropdown zur Auswahl der Rastergröße. Verfügbare Stufen: **4 · 8 · 16 · 24 · 32 · 64 px**. Standardwert ist 8 px (wie bisher).

Das Dropdown ist im inaktiven Zustand leicht ausgeblendet (Opacity 45 %) und wird beim Aktivieren des Grids zusammen mit dem Button hervorgehoben. Eine Änderung der Größe während das Grid aktiv ist, wird sofort live übernommen.

### Zoom – erweiterter Bereich bis 150 %

*(bereits in v1.1.6 eingeführt)* Der Zoom-Slider reicht jetzt von 25 % bis 150 % in 5 %-Schritten.
